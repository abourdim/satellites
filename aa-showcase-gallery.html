03 — Islamic Astronomy Heritage

Nine Scholars, Living Connections

Heritage is not a museum tab. Each scholar appears contextually when their contribution is relevant — al-Battani when the ISS passes, Ibn al-Haytham when an imaging satellite is visible.

903–986 CE

Al-Sufi

عبد الرحمن الصوفي

Book of Fixed Stars. First Andromeda observation. → Satellite star tracker catalogs.

780–850 CE

Al-Khwarizmi

محمد بن موسى الخوارزمي

Father of Algebra & Algorithms. → SGP4 algorithm name origin.

965–1040 CE

Ibn al-Haytham

ابن الهيثم

Book of Optics, scientific method. → Satellite camera optics.

973–1048 CE

Al-Biruni

أبو ريحان البيروني

Earth radius (16 km accuracy). → Geodesy for satellite orbits.

858–929 CE

Al-Battani

البتاني

Refined trigonometry. → Orbital period calculations.

1201–1274 CE

Nasir al-Din al-Tusi

نصیر الدین طوسی

Tusi couple, Maragheh Observatory. → Circular motion in orbits.

1394–1449 CE

Ulugh Beg

الغ بیگ

1,018-star catalog, Samarkand. → Star tracker databases.

1029–1087 CE

Al-Zarqali

الزرقالي

Universal astrolabe. → The original satellite tracker.

810–887 CE

Abbas ibn Firnas

عباس بن فرناس

First flight (875 CE), planetarium. → Aviation & space heritage.

04 — Feature Highlights

What Makes This Special

Not just another satellite tracker. This is a complete learning experience wrapped in a beautiful interface.

🌍

Real-Time Propagation

SGP4 algorithm computes satellite positions from NORAD TLE data — the same math used by mission control.

📱

Gyroscope Window

Point your phone at the sky. The display follows. Stars and satellites appear where they actually are overhead.

📻

Doppler Calculator

Live Doppler frequency shift calculated from radial velocity. Blueshift when approaching, redshift when receding.

💥

Kessler Simulation

Animated cascade showing how one collision creates debris that triggers more collisions — the nightmare scenario.

☽

Living Heritage

Scholars appear contextually. Al-Battani when ISS passes. Ibn al-Haytham when an imaging satellite is visible. Not a separate tab.

⚙️

Astrolabe Comparison

Side-by-side: what al-Zarqali's astrolabe did vs what this app does. Same 5 functions, 1,000 years apart.

📡

Frequency Bands

See which radio bands each satellite uses: GPS L1, Starlink Ku-band, ISS S-band. The invisible signals filling the sky.

💾

Data Export

One-click CSV or JSON export of every visible satellite with altitude, lat/lon, speed, inclination, type.

🗺️

Ground Track Map

Equirectangular world map showing satellite positions, color-coded by type, with observer location marker.

05 — Technical Foundation

Built to be Lightweight

Single HTML files. No build steps. No frameworks. No dependencies beyond satellite.js. Open any file in a browser and it works.

Frontend

Pure HTML/CSS/JS. Canvas rendering. CSS animations. No React, no Vue, no build tools. Single file architecture.

Satellite Data

CelesTrak TLE feeds (visual + stations). satellite.js for SGP4/SDP4 propagation. NORAD two-line elements.

Rendering

Canvas 2D with device pixel ratio scaling. Radial gradients, glow effects, orbit trails with time-based decay.

Sensors

Device Orientation API for gyroscope. Geolocation API for GPS. Touch/pointer events for interaction.

Calculations

Greenwich Sidereal Time. RA/Dec → Az/El conversion. Stereographic projection. Doppler shift at L-band.

Data

25+ satellite pattern matchers. 9 scholar objects. 9 Arabic stars. 8 modern space programs. Export via Blob API.
